\section{Exercises}


\begin{exercise}
When you use \java{printf}, the Java compiler does not check your formatting string.
See what happens if you try to display value with type \java{int} using \java{\%f}.
And what happens if you display a \java{float} using \java{\%d}?
What if you use two format specifiers, but then only provide one value?
\end{exercise}
